

## 0、勾股定理：根定理

- 作用：描述任何直角三角形三边关系
- 抽象公式，不带坐标
- 表达斜边和两直角边之间的普遍关系

```math
c^2 = a^2 + b^2
```

## 1、计算两点之间的距离
- 作用：在坐标系中求两点之间距离
- 是勾股定理的直接推导结果
- 把 a、b 换成了坐标差：
    - a = x₂ - x₁
    - b = y₂ - y₁

```math
d = √((x₂ - x₁)² + (y₂ - y₁)²)
```

```js
const dx = x2 - x1
const dy = y2 - y1
const distance = Math.sqrt(dx * dx + dy * dy)
```

## 2、斜边方向、45° 坐标分解

- 作用：45° 对应等腰直角三角形，x 与 y 分量完全相等
- 原理：是勾股定理 在特殊角度（a=b）下的特例公式
- 用途：所有需要“对角方向”或“45° 分量计算”的场景, 常用于:
    - 圆形或矩形的对角方向点计算
    - 对角线方向坐标计算
    - SVG 或 Canvas 中绘制 45° 斜线
    - 任意图形中沿 45° 方向的移动
    - 方向键、摇杆等 45° 输入判断（对角方向）

```math
sin(45°) = cos(45°) = 1/√2
```

在 45° 方向上，任意长度 d 的 x、y 分量均为：
```math
d/√2=d×sin(45°)=d×cos(45°)
```
你想沿 45° 方向移动多远，d 就是多少
